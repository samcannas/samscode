This directory stores verbatim upstream license texts for third-party material that is copied, imported, or adapted into Sam's Code.

- Keep `LICENSE` for Sam's Code's own MIT license.
- Add one file per upstream source when material from that source is incorporated into the repository.
- Use `NOTICE.md` to map each upstream notice to the local paths it covers.
- If a subtree already keeps a local `LICENSE.txt`, it may stay in place for convenience, but the root notice inventory should still be updated.
