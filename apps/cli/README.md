# samscode

Official helper CLI for Sam's Code desktop releases.

## Usage

- `npx samscode`
- `samscode`

The helper detects your operating system, prints the latest Sam's Code version it knows about, links to the GitHub releases page, and shows the best direct desktop download for your platform when available.

## Commands

- `samscode`
- `samscode desktop`
- `samscode install`
- `samscode release`
- `samscode --version`

## Desktop app

Sam's Code desktop builds are published on GitHub Releases:

- https://github.com/samcannas/samscode/releases
