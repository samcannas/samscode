# Sam's Code

Sam's Code is a desktop and web GUI for coding agents, with first-class support for Codex and Claude.

## Quick Start

- Install the CLI package with `npx samscode`
- Or download the desktop app from the GitHub releases page

## Development

- `bun install`
- `bun run dev` for the web/server workflow
- `bun run dev:desktop` for the desktop workflow

## Notes

- This project is still early and may change quickly
- Before finishing changes, run `bun fmt`, `bun lint`, and `bun typecheck`
